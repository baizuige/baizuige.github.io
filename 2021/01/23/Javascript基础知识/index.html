Javascript

 写这篇文档的原因是我又发现了一个宝藏，后盾人的向军，大叔的几个视频，帮助我很快的理解了Symbol、Map、Set，也因为大叔那优雅的代码风格，于是我准备再次重新学习JS，并记下笔记，防止以后又重新学。

第一章 走进Javascript

  • 在直接使用变量name的时候出现问题要知道可能是跟窗体有关
  • 变量提升：对于var声明的变量会在解析的时候提升到代码前面，但不赋值；let const会进行解析，但有临时性死区，更合理。
  • 虽然在声明变量的时候可以不写关键字，但这样会污染全局，可以使用严格模式避免
  • 过去使用立即执行函数来控制var的作用域。
  • var声明的变量会放到window对象中。
  • Object.freeze(obj)冻结对象中的属性，在严格模式下修改属性会报错
  • 传值和传址 值类型和引用类型
  • 写JS前先加上"use strict",规范自己的代码，也防止被糟糕的代码恶心

第二章 运算符与流程控制

ps：绝大多少语法都是和C/C++相通的

  • ==会进行形式转换，值相等即可；===为绝对等于，包含类型检测。

  • 或运算符可以用于赋值。

    let a = 3, b = 0;
let c = b || a; //c=3;

  • switch中的case可以使用表达式。

    let a = 3;
switch(true){
    case a>1:
        console.log("a>1");
        break;
    case a<1:
        console.log("a<1");
        break;
} //a>1

  • 子层循环想退出父层循环可以使用 break:标签，类似于goto，但不建议使用。

  • for-in取的是索引或键名，for-of取的是值。

第三章 值类型使用

  • typeof打印变量类型，数组和对象都是Object，未声明和未赋值都是undefined。

  • 可以使用instanceof判断是否为数组 //(arr instanceof Array)。

  • 标签模板，strings表示模板字面量中的字符串，lets表示模板字面量中的变量

    function links(strings, ...lets){
    console.log(strings); //["name:", ",age:", ".", raw: Array(3)]
    console.log(lets); //["baizuige", 21]
}
links`name:${user.name},age:${user.age}.`; //调用函数

    字符串常用属性 length、函数toUpperCase() toLowerCase() trim()

  • 字符串截取 slice() substring substr

    • slice(n[,m])截取从n到m的字符，不包含m，支持负数
    • substring(n[,m])截取从n到m的字符，不包含m，负数记为0
    • substr(n[,m])截取从n开始的m个字符，n支持负数，m负数记为0

  • 字符串检索 indexOf() includes() lastIndexOf() startsWith() endsWith()

    • indexOf(str[,n])从n位置开始向后查找str，找到则返回第一个字符的下标，没有则返回-1
    • includes(str[,n])从n位置开始向后查找str,找到则返回true，没有则返回false

  • 字符串替换 replace("old","new")

  • 字符串重复 repeat(n) //重复n次

  • 类型转换 ps：不全

    • 字符串转为数值 Number(str)、+str
    • 数值转为字符串 String(num)、num+""
    • 转为布尔 !!let、Boolean(let)
    • parseInt()取字符串前面的数值
    • split("")字符串拆分为数组

  • 数值类型除了0都是真，字符串类型除了空字符串都为真、数组、对象在使用Boolean()作类型转换时总是为真

  • toFixed(n)浮点数四舍五入并保留n位小数

  • Math中常用函数min()、max()、ceil()、floor()、round()

  • Math.random()取随机数、取值范围[0,1)

  • 日期类型 new Date()

    • 年getFullYear()、月getMonth()、日getDate()、星期getDay()、时getHours()、分getMinutes()、秒getSeconds()
    • 获取时间戳date.valueOf()和+date
    console.time("for");
for(let i = 0; i < 1000; i++){}
console.timeEnd("for"); //for: 0.048828125 ms

  • 日期处理库 Moment.js

第四章 数组

ps：引用类型，可以存储对象

  • new Array(6)创建6个空元素

  • Array.from(str)就可迭代对象或类数组对象转为数组 //ps:不完整

  • [...arr]展开数组或收集

  • 解构语法

    let arr = ["baizuige", 21];
let [name,age] = arr; //name = "baizuige"; age = 21;
    let a = 2, b = 3;
[b, a] = [a, b]; //a = 3; b = 2;

    数组方法备忘单：

    • 添加/删除元素：
      • push(...items) —— 向尾端添加元素，
      • pop() —— 从尾端提取一个元素，
      • shift() —— 从首端提取一个元素，
      • unshift(...items) —— 向首端添加元素，
      • splice(pos, deleteCount, ...items) —— 从 pos 开始删除 deleteCount 个元素，并插入 items。
      • slice(start, end) —— 创建一个新数组，将从索引 start 到索引 end（但不包括 end）的元素复制进去。
      • concat(...items) —— 返回一个新数组：复制当前数组的所有元素，并向其中添加 items。如果 items 中的任意一项是一个数组，那么就取其元素。
    • 搜索元素：
      • indexOf/lastIndexOf(item, pos) —— 从索引 pos 开始搜索 item，搜索到则返回该项的索引，否则返回 -1。
      • includes(value) —— 如果数组有 value，则返回 true，否则返回 false。
      • find/filter(func) —— 通过 func 过滤元素，返回使 func 返回 true 的第一个值/所有值。
      • findIndex 和 find 类似，但返回索引而不是值。
    • 遍历元素：
      • forEach(func) —— 对每个元素都调用 func，不返回任何内容。
    • 转换数组：
      • map(func) —— 根据对每个元素调用 func 的结果创建一个新数组。
      • sort(func) —— 对数组进行原位（in-place）排序，然后返回它。
      • reverse() —— 原位（in-place）反转数组，然后返回它。
      • split/join —— 将字符串转换为数组并返回。
      • reduce/reduceRight(func, initial) —— 通过对每个元素调用 func 计算数组上的单个值，并在调用之间传递中间结果。
    • 其他：
      • Array.isArray(arr) 检查 arr 是否是一个数组。

    请注意，sort，reverse 和 splice 方法修改的是数组本身。

    这些是最常用的方法，它们覆盖 99％ 的用例。但是还有其他几个：

    • arr.some(fn)/arr.every(fn) 检查数组。

      与 map 类似，对数组的每个元素调用函数 fn。如果任何/所有结果为 true，则返回 true，否则返回 false。

      这两个方法的行为类似于 || 和 && 运算符：如果 fn 返回一个真值，arr.some() 立即返回 true 并停止迭代其余数组项；如果 fn 返回一个假值，arr.every() 立即返回 false 并停止对其余数组项的迭代。

      我们可以使用 every 来比较数组：

      function arraysEqual(arr1, arr2) {
  return arr1.length === arr2.length && arr1.every((value, index) => value === arr2[index]);
}

alert( arraysEqual([1, 2], [1, 2])); // true

    • arr.fill(value, start, end) —— 从索引 start 到 end，用重复的 value 填充数组。

    • arr.copyWithin(target, start, end) —— 将从位置 start 到 end 的所有元素复制到 自身 的 target 位置（覆盖现有元素）。

    • arr.flat(depth)/arr.flatMap(fn) 从多维数组创建一个新的扁平数组。

    • [arr.of(element0[, element1[, …[, elementN]]])](https://developer.mozilla.org/zh-CN/docs/Web/JavaScript/Reference/Global_Objects/Array/of)基于可变数量的参数创建一个新的 Array 实例，而不需要考虑参数的数量或类型。

第五章 Symbol、Set和Map

Symbol类型

  • Symbol()创建一个Symbol类型
  • Symbol.for()在全局符合注册表中创建并重用符号
  • Symbol.keyFor()查询全局注册表
  • Object.getOwnPropertySymbols()遍历Symbol属性
  • Reflect.ownKeys()遍历所有属性

Set和WeakSet类型

ps：去重的数组

  • 通过new声明Set类型时可以在（）中包含要存入的元素
  • add()向Set类型变量添加元素、size获取元素个数、has()判断元素是否存在、delete()删除元素,没有则返回false、values()获取变量中的元素、clear()删除变量中的所有元素
let set = new Set("baizuige"); //Set(7) {"b", "a", "i", "z", "u", …}
let set = new Set(["bai","zui","ge"]); //Set(3) {"bai", "zui", "ge"}
set.add(1);
set.size;
set.has("bai"); //ture
set.delete("zui");
set.values();
set.clear();
  • 由于第一个特性可以简单的将一个字符串拆分为数组，并执行一些操作
let a = "123456789";
let set = new Set(a);
let arr = [...set].filter(function(item){
	return item < 5;
});
console.log(arr); //(4) ["1", "2", "3", "4"]
  • 由于Set中会消除重复的元素，所有可以用来去重。
let a = [1,2,3,4,5,4,3,2,1];
console.log(new Set(a)); //Set(5) {1, 2, 3, 4, 5}
console.log([...new Set(a)]); //(5) [1, 2, 3, 4, 5]
  • WeakSet中存放的是引用类型，且为弱引用，不能进行遍历，当引用的元素删除后不会保存引用。 ps：不了解使用场景，大概是为了不浪费资源。

Map和WeakMap类型

ps：更强大的对象

  • Map可以将任何类型作为键名，不像对象只能使用字符串作键名
  • set(,)像Map类型变量添加元素、size获取元素个数、has()判断元素是否存在、delete()删除元素,没有则返回false、keys()获得变量中元素的键、 values()获取变量中元素的值、entries()获得变量中元素的键值对、clear()删除变量中的所有元素
let map = new Map([["name","baizuige"],["age","21"]]); //Map(2) {"name" => "baizuige", "age" => "21"}
let map = new Map();
map.set("name","baizuige");
map.set(function(){},"function()");
map.set({},"array");
map.set(1,"number");
  • 由于Map的键名可以是对象，那么可以用Map来存储DOM对象
  • WeakMap类型的键必须是对象，同时也是弱引用类型，不能进行遍历

第六章 函数

  • 函数声明会产生函数声明提升，即可以在函数声明之前调用函数；同时会保存在window对象中

  • 函数表达式不能在函数声明前调用函数，也不会保存在window对象中

  • 立即执行函数

    (function(){
	console.log("Hello World!");
})();

  • ​ arguments保存函数的所有参数，类数组，老语法，现在可以直接使用点语法

  • 作为参数传递跟另一个函数的函数叫回调函数

  • 函数中的this是window对象，方法中的this是当前对象；方法中可以将this赋值给变量让其内部的函数调用当前对象，有些函数支持第二个参数来修改当前函数中的this

  • 箭头函数的this在运行时绑定到执行函数的上下文

  • func.call(context,...args)设置函数的this为context，并调用函数

  • func.apply(context,args)设置函数的this为context，并调用函数，args为类数组对象

  • func.bind(context,arg1,arg2,...)设置函数的this为context，返回一个新的函数，arg的参数优先级大于调用新函数时参数的优先级
