Window下Hexo建站

2021-01-23 笔记 525 字
Hexo Window

建站准备

  • 安装git
  • 安装node.js，长期支持版即可

实操部分

安装完git后鼠标右键可以看到Git Bash Here操作，打开后可以使用一些linux的命令；选择自己想要存放博客文件的文件夹内，执行该操作，之后的操作都在这里执行。

  1. 安装并切换阿里镜像

    # npm install cnpm -g --registry=https://registry.npm.taobao.org
# npm config set registry https://registry.npm.taobao.org

  2. 安装hexo

    # npm install hexo-cli -g

  3. 初始化一个hexo博客

    # hexo init

    此时博客已经成功生成，可以使用hexo s在本地预览

部署远端

这里推荐部署在github上，免费且方便；也可以部署在gitee上，同样是免费的还是在git命令行中执行以下操作

  1. Git初始化配置

    # git config --global user.name "填写自己github的用户名"
# git config --global user.email 填写绑定的邮箱号

  2. 进入github，创建一个以”自己用户名.github.io”为名称的仓库

  3. 复制仓库地址，HTTPS和SSH地址都行

  4. 配置hexo

    1. 在博客文件夹中打开_config.yml文件

    2. 翻到最后，输入以下代码，其中repo后输入刚才复制的地址

      # Deployment
## Docs: https://hexo.io/docs/one-command-deployment
deploy:
  type: git
  repo: 仓库地址
  branch: main

  5. 回到git命令行，输入hexo d即可将博客部署到远端；部署时可能会出现错误ERROR Deployer not found: git 解决办法是安装hexo-deployer-git插件

    # npm install --save hexo-deployer-git

  6. 回到github仓库，在设置中找到GitHub Pages，将Source中的Branch改为main后保存(save)，之后就可以在”自己用户名.github.io”的网址下访问博客

常用命令

  • hexo new "文章名称" 在\source\_posts目录下创建一篇文章，markdown格式书写，推荐使用typora
  • hexo g 生成静态文件
  • hexo s 本地预览
  • hexo d 推送远端、hexo d -g 部署之前预先生成静态文件
  • hexo clean 清除缓存文件和已生成的静态文件，更换主题后可能用到
